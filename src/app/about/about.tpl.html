What is it?

This web app is designed to help you wake up with the sounds you like. They may be songs, mixes, playlists, recorded radio shows or live web radio stations!
Setting an alarm is as easy as choosing the time and pasting the link of a website. You can manage a list of alarms which will still be there when you come back another time.
This app can be used on a computer with speakers or a mobile; you only need to keep the browser opened.

Allowed links

All links are allowed! You can copy and paste links from Youtube, TuneIn, SoundCloud, DI.fm and many more! You can use links to local files too.
Just make sure that a sound is played directly when the window is opened.

Pop-up policy

Please allow pop-ups in order to open the links when time has come. Nowadays, browsers block pop-ups by default to prevent from SPAMs, undesired commercial content... You can test the permissions of your browser by setting an alarm in one minute. If there is an alert, please choose to "Allow pop-ups on this site".
Pop-up ads will never come.
